VERIFICA INSTALLAZIONE PORTALE NPM

l'npm è un portale contenente diversi moduli già pronti e utilizzabili per i nostri progetti

Verificare che esso sia installato eseguendo il seguente comando nel terminale:

						
npm -v

PROBLEMA VERSIONE Node JS

Potrebbe succedere che durante la creazione di un progetto il terminale restituisca un errore riguardante la versione di Node JS

                        
You are running Node 12.22.9.

Create React App requires Node 14 or higher.
Please update your version of Node.

Qui di seguito è riportata la procedura per correggere l'errore

						
sudo npm install -g n
(Installa il comando "n")

sudo n stable
(Installa l'ultima versione stabile di Node JS)

Dopo aver eseguito questi 2 comandi assicurarsi di chiudere il terminale e riaprilo, a questo punto sarà possibile creare il nuovo progetto
