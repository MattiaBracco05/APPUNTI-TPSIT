Variabili d'ambiente

Le variabili d'ambiente vengono utilizzate per nascondere il token, per nascondere la porta (Node JS) o per nascondere altre informazioni

Esempio lettura di una variabile

All’interno della cartella root creo un file di testo .env (file nascosto) dove vado a dichiarare le variabili nel seguente modo:
N.B. il file .env deve essere creato dall'editor di testo perchè andandolo a creare direttamente nella cartella dal momento in cui diamo invio il file .env viene nascosto e pertanto non è possibile aprirlo

File .env

                        
REACT_APP_KEY-”A1B2C3DE5F6G7”

Per recuperare una variabile d'ambiente è necessario utilizzare il seguente codice:
N.B. il sito deve essere riavviato dopo aver scritto questa riga di codice altrimenti la variabile non viene mostrtata

File App.js

                        
import React from 'react'

const App = () => {

    //RETURN --> Stampa a video
    return (
        <div>
            <h1>Variabili d'ambiente</h1>
            Variabile d'ambiente nascosta: {process.env.REACT_APP_KEY}
        </div>
    )
}

export default App
