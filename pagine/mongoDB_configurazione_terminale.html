CONFIGURAZIONE DA TERMINALE

In questa pagina è spiegato come creare, visualizzare e modificare un database tramite interfaccia a linee di comando

						

····················DATABASE····················
VISUALIZZO I DB ESISTENTI
show dbs

SELEZIONO UN DB (o lo creo in caso non esistesse)
use nomeDB

ELIMINO IL DB SELEZIONATO
db.dropDatabase()

····················COLLECTION····················
VISUALIZZO LE COLLECTION (informazioni) PRESENTI IN UN DB
db.getCollectionNames()

SELEZIONO UNA COLLECTION (o la creo in caso non esistesse)
db.utenti

INSERISCO DATI (in formato JSON) IN UNA COLLECTION (metodo deprecato!)
db.utenti.insert({nome:"Mario", cognome:"Rossi"})

INSERISCO UN OGGETTO NELLA COLLECTION
db.utenti.insertOne({nome:”Mario”, cognome:”Rossi})

INSERISCO PIÙ OGGETTI (array) NELLA COLLECTION
db.utenti.insertMany([{nome:”Mario”, cognome:”Rossi”}, {nome:"Luca", cognome:"Bianco"}])

VISUALIZZO I DATI DI UNA COLLECTION
db.utenti.find()

FILTRO I DATI DA VISUALIZZARE DI UNA COLLECTION
db.utenti.find({cognome:”rossi”})

ELIMINO UNA COLLECTION
db.utenti.drop()

ELIMINO UN ELEMENTO IN UNA COLLECTION
db.ordini.deleteOne({articolo:”pizza”})
