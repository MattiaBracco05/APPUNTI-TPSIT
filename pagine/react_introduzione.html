REACT

È una libreria che ci permette di creare dei siti web realizzati su un'unica pagina composta da diversi componenti. I siti web realizzati con react si aggiornano in background pertanto non necessitano di dover ricaricare la pagina (un esempio di stio realizzato con react è il social network Facebook)
SPA (Single Page Application) ovvero applicazioni realizzate su una pagina singola
Il principale vantaggio di react è quello di poter importare dei moduli per esempio per la gestione di PDF o la realizzazione di grafici React è consigliato per la realizzazioni di siti web con il fine di gestire: registri, magazzini, ordinazioni e social network
