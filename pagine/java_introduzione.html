JAVA

Java è un linguaggio di programmazione ad alto livello, orientato agli oggetti e progettato per essere indipendente dalla piattaforma (un programma scritto in Java può essere eseguito su qualsiasi dispositivo che sia dotato di una Java Virtual Machine (JVM)).
Java è stato sviluppato da Sun Microsystems, oggi divisione di Oracle Corporation.

La JVM

Jave gira su una JVM (Java Virtual Machine) dopo essere stato compilato per l’OS, Java compilato per Android ha l’estensione .apk
