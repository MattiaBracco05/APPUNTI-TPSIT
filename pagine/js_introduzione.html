Javascript

È un linguaggio di programmazione utilizzato per realizzare pagine web interattive

Per cosa è utilizzato?

Ecco alcune cose che possiamo fare inserendo JavaScript all'interno di un sito web:
· Aggiornare il feed e modificare il layout rendendolo "dinamico"
· Visualizzare animazioni e mappe interattive
· Interagire con l'utente inserendo dei valori in input
· Interagire con l'utente inserendo messaggi in output
· Effettuare calcoli e operazioni matematiche
